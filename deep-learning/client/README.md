
## Prérequis
- Node.js 19+
- npm ou yarn

## Frontend (React + TypeScript + Vite)

1. **Se placer dans le dossier client**
   ```bash
   cd /client
   ```

2. **Installer les dépendances**
   ```bash
   npm install
   ```

3. **Lancer le serveur de développement**
   ```bash
   npm run dev
   ```
   
   L'application sera accessible sur `http://localhost:5173`


## Utilisation

1. Assurez-vous que le backend est lancé en premier
2. Lancez ensuite le frontend
3. Ouvrez votre navigateur sur `http://localhost:5173`
4. Remplissez le formulaire avec un titre de projet et un budget
5. Cliquez sur "Analyser le projet" pour obtenir la prédiction
